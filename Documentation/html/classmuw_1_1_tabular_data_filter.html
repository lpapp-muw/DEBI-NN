The TabularDataFilter class is a convenience implementation of several filtering steps to manipulate or extract information from TabularData objects.

Identifies the common keys of the feature and label databases as of the current label.

Parameters
[in]aFeatureDatabasethe tabular dataset containing the feature vectors.
[in]aLabelDatabasethe tabular dataset containing the label vectors.
[in]aLabelIndexthe column index in the label database to take the current label subgroups from. Shall be 0 in case only one label column is in the label database.
[in]aIsNAIncludedif true, keys having missing values as of aLabelIndex will be part of the common key identification process. If false, keys corresponding to missing label values will be skipped.
Returns
The common keys present in both feature and label databases.
